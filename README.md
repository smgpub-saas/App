# SMGPUB Platform

A full-stack publication platform with Stripe integration.

## 🚀 Quick Start
1. Run `npm run install-all` to set up both nodes.
2. Add your Stripe keys to `server/.env`.
3. Run `npm run dev:client` and `npm run dev:server` to start.

## 🏗️ Architecture
- **Frontend:** Vite (Deployed to Vercel)
- **Backend:** Node.js/Express (Deployed to Replit/Render)
- **Domain:** GoDaddy (smgpub.com)
